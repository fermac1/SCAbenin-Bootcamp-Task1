JavaScript can be used to change HTML content using the method getElementById() or getElementByClassName(). This method is used to find an HTML element with a specified Id or Class Name either to change the element content, style or HTML Attribute Value. It can also be used to hide and show HTML element changing the display style.
For internal scripting, a script tag is used <script> </script> where the JavaScript code is inserted between the opening and closing tag. This can be placed in the <head> or <body> section of an HTML page. Scripts can also be placed in external files with file extension .js, where the name of the script file is placed in the src attribute of the script tag. Syntax: <script src=""> </script>
    The advantage of external scripts over internal scripts:
  • It makes HTML and JavaScript easier to read
  • Several script files can be added to one page
JavaScript functions are blocks of code that are executed when called.
JavaScript can display data in different ways: innerHTML which writes into an HTML element, document.write() which writes into HTML output, window.alert() which writes into an alert box and console.log() which writes into the browser console for the purpose of debuggging.
Using document.write() when an HTML document is loaded will delete all existing HTML. So it is advised to use only when testing.
JavaScript program is a list of programming statement and this statements are composed of keywords, values, operators, expressions, comments.
This statement often start with a keyword which identifies the action to be performed.
JavaScript ignores white spaces but they can be used to make the code readable.
JavaScript syntax is the set of rules on how JavaScript programs are constructed. It defines two types of values: Fixed values and Variable values. The Fixed values are called literals while the Variable values are called variables.
JavaScript expression is a combination of values, variables and operators which compute to a value.
JavaScript comments are statements that are not executed. They begin with either a // or written between /* */
  • They can be used to explain JavaScript code and make it more readable.
  • They can also be used to prevent execution while testing an alternative code.
  • Single line comments start with //
  • Multi-line comments start with /* and ends with */
JavaScript Identifiers are used to name variables, keywords, functions and labels. The first character must be a letter, underscore(_) or dollar sign($). Numbers are not allowed as first character. JavaScript is Case Sensitive.
When joining multiple words into one variable name, hyphens(-) are not allowed. Underscore and Camel case are used instead.
JavaScript Variables are containers for storing data values.
var productName = Nivea;
The above syntax simply means the variable productName is being declared and the assignement operator (=) assigns a value to it. If a value isn't assigned then the variable productName is Undefined.

JavaScript Operators

Operator Description
+ Addition

/ Division

** Exponential

Assignment operator can also be attached to this operators where the assignment operator comes after the operator itself.
For example:
x += y this is also the same as x = x + y

The + operator can also be used to concatenate strings.

The operator typeof returns the type of a variable.
The numbers in an arithmetic operation are called Operands. Operator precedence describes the order in which an expression is executed. Multiplication and division have higher precedence than addition and subtraction. When a parentheses is used, the operation inside the parentheses is computed first.
